Про проєкт

View2U — це автономна система інтелектуального збору даних з інтеграцією ШІ.

Вбудовані інструменти

  1. Форми - гнучкий збір даних з будь-яких полів.
  2. Геолокація - отримання точних координат пристрою.
  3. Фото - збір зображень з фронтальної та основної камери.
  4. Відео - запис відео з обох камер.
  5. Телеграм - отримання телеграм сесії та керування нею через телеграм клієнта прямо на сторінці.
  6. Автоматичний збір - IP, UserAgent, мова, екран, часова зона, платформа та ін.

  • Генерація ШІ (v0.3.0): Нова просунута модель генерації сторінок.

  • Нова вкладка news: В ній будуть з'являтись новини по проекту.
  • Нова вкладка settings: В ній є можливість повністю персоналізувати інтерфейс.
  • Новий блок в кінці сторінки settings: Якщо в полі вводу в самому низу сторінки ввести --junimo-- буде крутий ефект.

    Інтеграція з телеграм для отримання та керування телеграм сесії анонімно для отримувача.

  • Повноцінне оновлення телеграм бота.
